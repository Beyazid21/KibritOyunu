# KibritOyunu
Kibrit Oyunu — klassik zeka oyunu olaraq iki nəfər arasında oynanır.
Oyun qaydaları çox sadədir: məqsəd son kibriti götürməməkdir.
Son kibriti götürən oyunçu məğlub olur.

Qayda 1:
Bir sətirdən istənilən sayda kibrit götürmək olar (amma yalnız bir sətirdən).

Qayda 2:
Bir oyunçu kibrit götürdükdən sonra növbə qarşı tərəfə keçir.

Qayda 3:
Bütün kibritlər bitdikdə, son kibriti götürən oyunçu məğlub sayılır.
